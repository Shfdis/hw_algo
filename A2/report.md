# Отчет A2

## Методы

- Стандартный MERGE SORT
- Гибридный MERGE+INSERTION SORT

## Параметры

- Размеры: 1000-50000, шаг 2000
- Типы: случайные, обратно отсортированные, почти отсортированные
- Threshold: 10, 15, 20, 30
- Замеров: 5

## Выводы

1. Гибридный алгоритм эффективнее на небольших массивах
2. Оптимальный threshold: 15-20 для случайных данных
3. При больших размерах (>50000) стандартный может быть быстрее
